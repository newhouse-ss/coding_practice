This repository tracks my continuous learning and technical preparation for Software Engineering and Data Science roles. It is structured into three core modules:

## 📂 Repository Structure

- **`dsa/` (Data Structures & Algorithms)**
  - Implementations of fundamental algorithms and solutions to competitive programming problems (LeetCode, Codeforces, etc.).
  - Focus on time/space complexity optimization.
- **`ml/` (Machine Learning Coding)**
  - From-scratch implementations of ML algorithms, neural network architectures, and common MLOps patterns.
  - Focus on PyTorch, NumPy, and efficient data processing.
- **`live-mock/` (Mock Interview Practice)**
  - Records and post-mortems of live coding sessions and technical mock interviews.
  - Focus on problem-solving communication and whiteboarding.
